gits
====
git wrapper

Usage:
------
`gits`  
list accept git commands

`gits -template > /path/to/your/watchlist.json`

edit watchlist.json, add your repository

`gits -conf /path/to/your/watchlist.json [accept git command]`


Requirements:
-------------
`git`

Install:
--------
`go get github.com/kamisari/gits`

or
